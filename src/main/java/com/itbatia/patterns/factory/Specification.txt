
Шаблон - Фабрика (Factory method)
___________________________________________________________________________________

Цель:

Создание интерфейса, который создаёт объект. При этом, выбор того, экземпляр какого
класса создавать остаётся за классами, которые имплементируют данный интерфейс.
___________________________________________________________________________________

Для чего используется:

Для делегирования создания экземпляров другому классу
___________________________________________________________________________________

Пример использования:

- заранее не известно - экземпляры какого класса нужно будет создавать;
- класс спроектирован таким образом, что создаваемые им объекты имеют свойства
  определённого класса.
___________________________________________________________________________________
